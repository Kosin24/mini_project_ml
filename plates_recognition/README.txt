Этот проект содержит инструменты для сравнения предсказаний на "чистых" и "грязных" версиях данных. 
https://www.kaggle.com/code/vladsamarov/prediction-analysis-cleaned-vs-dirty-v2/input

Файлы в проекте:
Arg.ipynb - Основной ноутбук с анализом данных и сравнением результатов

plates_rec.cipynb - Вспомогательный ноутбук для распознавания номерных знаков

masks.zip - архив с масками для обработки изображений

submission.csv - файла с предсказаниями модели
